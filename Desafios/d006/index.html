Tags em HTML

Em HTML5, podemos inseriros elementos da lista a seguir.
Passe o Mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Parágrafo
  • Títulos
    • Subtítulos
  • Imagens
    • insira a imagem, pressione CTRL+ESPAÇO e escolha
    • insira uma descrição da imagem
  • Abraviações
    • Descrição daquele texto
  • Códigos-fonte
  • Listas numeradas
    • crie uma sequência de números, letras ou algarismos
    • Sequência que preferir:
      1. Números
      1. Letras
      1. Algarismos
  • Listas com demarcadores
    • Simbolos para demarcação da lista:
      • Quadrado
      • Círculo
      • Disco
