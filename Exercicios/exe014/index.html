Capítulo 1.1

Ao invés de modificar as CSS em cada linha, usa-se a tag <style> como uma das configurações da <head>

Dentro desta tag, cria-se seletores. Por exemplo:

<style>

body {

background-color="aqua;"

font-family="cursive;"

font-size="20px;"

}

h1{

background-color="aliceblue;"

color="black;"

}

h2 {

color="crimson;"

}

p {

text-align="justify;"

}

</style>
