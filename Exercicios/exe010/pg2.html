Entendendo links internos

Para criar um acesso à segunda página é possível criar um link pela primeira página.

Usando a tag "a" no "href" clique crtl+espaço e escolha o link da página que quer relacionar.

Em "rel" escreva next se for a próxima página ou "prev" se for a página anterior.

Caso queira criar links patrocinados ou links não oficiais do seu material use o "rel" "nofollow".Exemplo
