Trabalhando com listas

Ordenadas

  1. Com o uso da tag <ol> cria-se listas ordenadas
  2. Com o uso da tag <li> cria-se o item de cada lista
  3. Com o uso do parametro "type" cria-se a forma de classificação das listas
  4. Em listas ordenadas, os tipos de classificações são
    1. A
    2. a
    3. I
    4. i
    5. 1

Não Ordenadas

  • Com o uso da tag <ul> cria-se listas não ordenadas
  • Com o uso da tag <li> cria-se o item de cada lista
  • Com o uso do parametro "type" cria-se a forma de classificação das listas
  • Em listas não ordenadas, os tipos de classificações são
    • disc
    • circle
    • square

Listas mistas

  1. - Uma alternativa para misturar as listas é:
    1. Criar sub-itens de maneira ordenada ou não ordenada
    2. Criar sub-itens que podem servir dividindo as informações, mas sem mexer na continuidade delas
      • Obs: Caso queira mudar mais de uma tag ao mesmo tempo, clique sobre as tags enquanto pressiona a tecla "Alt".
  1. Além de explicar outros fatores, o itens acima também serviram para demonstração
  2. A partir deles fica demonstrado como é possível ordenar listas mistas
    1. Obs: Para continuar a lista de onde parou, use a tag "Type" e depois a Tag "Start" dando continuidade a partir do número onde a lista parou.
