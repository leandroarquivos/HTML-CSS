Testando carregamento de imagens

Como usar imagens corretamente sem problemas de copy-right?

Google imagens - direitos de uso - marcadas para reutilização com modificiação (Creative Commons)

Estas imagens tem liberdade de uso e modificações na internet

Outras allternativas: pesuisar na "Unsplash" ou na "Pexels"

Observação: cuidado com as imagens nos sites citados, pois algumas não são de uso livre

Para dimensionar o tamanho das imagens: Abrir no GIMP - redimensionar imagem - definir largura (GIMP define a altura automaticamente.) - exportar como...

Observação: atenção para na forma salvar imagem redimensionada durante a exportação dela.

Obervação 2: Sites com imagens muito pesadas, demoram para carregar. Demandam muito tempo de processamento.
