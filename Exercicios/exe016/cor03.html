Exemplo

Título 1

Usando as CSS use tags e parametros base para criação de site

  1. "*" para definir padrões gerais das CSS na página
  2. "body" para definir padrões de tudo o que estiver no corpo da página, ou seja, tudo que aparece na página.
  3. "main" para definir padrões daquilo que aparece dentro do quadro escolhido
  4. "h1" para definir padrões de todos os títulos na página
  5. "h2" para definir padrões de todos os subtítulos na página
  6. "p" para definir padrões de todos os parágrafos na página

Título 2

Observações:

  1. O uso de chaves "{}" funciona nas CSS para abrigar os parametros definidos;
  2. Os parametros seguirão como prioridade, a ordem em que foram postos no código;
  3. Uso de "/**/" para comentários que não apareceram na página, apenas no código.

Título 3

Para saber os parametros das CSS, entre na CSS e leia os "/**/" presentes lá.
