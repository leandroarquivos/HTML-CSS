Capítulo 1

Capítulo 1.1 - Cor de plano de fundo

Para utilizar uma das CSS como, por exemplo, mudar a cor do fundo desta página, use o parametro style="background-color: ;" e escolha a cor.

Antigamente usava-se, para esta mesma função, a CSS "bgcolor", mas ela não é mais funcional.

tags e parametros que aparecerem em vermelho não são mais reconhecidas.

Capítulo 1.2 - Cor de linha de Texto

Para mudar a cor da fonte, como demonstrado acima, use style="color: ;" e escolha a cor

Capítulo 2

Capítulo 2.1 - Mais de uma CSS

Um exemplo disso é que pode usar apenas um parametro "style" com todas as CSS que achar necessário

style="background-color: blue; color: yellow; font-family: Arial".

Isso propõe que o plano de fundo será azul, o que for escrito será amarelo e, a forma da letra será em Arial.

As configurações Inline são para detalhes, questões pontuais, mas não deve ser usado para desenvolvimento do site por completo.
