Tags não utilizaveis

Como os códigos mudam o tempo todo e evoluem, as tags também evoluem. Algumas não estão mais em conformidade com o HTML como, por exemplo <font>. Elas são direcionadas à forma e não mais ao conteúdo.

Por isso é necessário ficar atento e revisar constantemente os conhecimentos de desenvolvimento web

Exemplo de Semântica do html5:

Antigamente usava-se a tag <u> para sublinhar palavras, mas isso tem mais uma relação com a forma do que com conteúdo. Mas atualmente, as informações são melhor didividas. O que é html lida com o conteúdo. um endereço ficar melhor identificado com a tag <adress> ao inves de apenas sublinhada.

Exemplo:
Vila Moraes, São Paulo - SP, 04171-010

Desta maneira endereço (conteúdo) pode acabar virando um link para encontra-lo em mapas online

Resumindo

html = significado = semântica = conteúdo
