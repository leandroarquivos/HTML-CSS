Principais formatações

Negrito ou Destaque

Nesta frase usando a tag <b>, uma tag não semântica (sem significado, apenas com a forma)

Nesta frase usando a tag <strong> uma tag semântica (visualamente não muda, mas muda no entendimento do significado para o programa e para desenvolvedores)

Itálico/Ênfase

Nesta frase usando a tag <i>, uma tag não semântica

Nesta frase usando a tag <em>, uma tag semântica

Texto Marcado / Marca-texto

podemos criar um texto marcado usando a tag <mark>

OBS:algumas tags podem não aparecer facilmente.Para resolver isto existe um facilitador. CTRL+SHIFT+P - Envelopar com abreviação (ou Wrap with abbrevition) e escreva o nome da formatação que deseja.

Para entender o CSS nestas formatações, dentro da formatação Mark eu posso mudar o seu estilo usando a formatação style+background color+uma corNeste caso a cor verde "lime"

A formatação <style> pode ser usada de 3 formas:

1. Definindo uma formatação de html unica (como no exemplo acima)

2. Definindo uma formatação de css para a página em si sendo descrito na <head>

3. Usando um link externo

Textos grandes e pequenos

Para letras grandes existe a tag <big> (não semântica) e para letras pequenas existe a tag <small> (Semântica)

Texto deletado

São textos que podem ser lidos, mas não devem ser mais considerados como participantes do todo.Estes textos aparecem riscados para dar este significado

Para isto usa-se a tag <del>

Texto inserido

São textos inseridos depois

Para isto usa-se a tag <ins> para mostrar no texto um destaque da informação.

Texto sobrescrito

Muito usual em matemáticas. Exemplo X ao quadrado. Usando a tag <sup> conseguimos organizar semântica assim: X2.

Texto subscrito

Muito usual em química. Exemplo a formúla da água. Usando a tag <sub> conseguimos organizar semântica assim: H2O
