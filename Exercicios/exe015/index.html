Capítulo 1

Capítulo 1.1

Cria-se uma página nova, mas ao invés de ser ".html" ela tem final ".css".

Capítulo 1.2

Dentro desta página não é preciso criar "head" nem "body", mas é aconselhável usar na primeira linha do código o texto: @charset "UTF-8";

Em cada página, na "head" (acima ou abaixo do <title>) insira um <link> com o href da página css que criou)

todos os estilos que forem criados na página das css aparecerão em todas as páginas de html com o link.

OBS: É possível criar mais de um css para cada página html. Só é preciso que seja inserido o link corretamente na "head" da página html.

Capítulo 2

Capítulo 2.1

O estilo externo é bem geral e funciona para dar o mesmo estilo a várias páginas ao mesmo tempo.

O estilo interno (ou estilo local) funciona bem para definir detalhes de cada página.
Aconselhável usar apenas para fatores pontuais de uma página específica, pois os detalhes deste estilo se sobrepõe ao externo.

O estilo inline funiciona para definir detalhes ainda mais minusciosos em cada linha do código.
Aconselhável usar para fatores muito pontuias pois, seus detalhes são sobrepostos às definições dos outros dois estilos.
